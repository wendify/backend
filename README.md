# backend

The backend for Wendify.

## Installation

Create a virtual environment:

```
python -m venv .venv
```

Start the venv:

- Unix / macOS

  ```
  source .venv/bin/activate
  ```

- Windows

  ```
  .venv\Scripts\activate
  ```

Install dependencies

```
pip install -e .
```

Copy & configure environment variables

```
cp .env.example .env
```

Run the main script to start the backend:

```
main
```
